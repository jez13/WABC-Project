The WABC Project

What is WABC?

WABC (Windows Application Builder Collection) is a program used to easily build programming applications.

What programming languages does WABC support?

Currently, WABC is in Pre-Alpha, so there aren't much programming languages.
WABC Supports:
  • Java
  • C (Work in progress)
  • C++ (Work in progress)

How do i use WABC?

For WABC to work, you need a build.cfg file.
build.cfg uses the TOML programming language to set the settings for WABC.
Example build.cfg for java: 
 
 [Project]
     Name = "HelloWorld"
     Compiler = "C:\\Program Files\\Java\\jdk-22\\bin\\javac.exe"
 [Manifest]
     File = "manifest.txt"
 [DirectoryTree]
     Source = "src"
     Build = "build"
     Distribution = "dist"
